# PhotoGama

Welcome to the PhotoGama page!  

## What is PhotoGama?

PhotoGama is a texture creator that I'll be using in my games  
The current versions are:   
Alpha 0.1:  
```
Support for brush
Support for bucket tool
Support for random noise
Support for blur
Support for save
```
